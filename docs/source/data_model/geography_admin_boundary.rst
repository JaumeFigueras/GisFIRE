Administrative boundary
=======================

The generic model for administrative divisions — countries, regions, provinces,
municipalities — which nest into a hierarchy of arbitrary depth. See
:doc:`../data_model` for how the nesting and the spatial lookups work, and
:doc:`../providers/ocha_admin_boundary` for the source GisFIRE imports countries from.
